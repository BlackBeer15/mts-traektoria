МТС Траектория (Фронт) - это веб-приложение на React JS, который помогает молодым людям выбрать профессию, построить карьерный трек и найти первую работу или стажировку.
Проект решает проблемы отсутствия профориентации, непонимания карьерного пути и сложностей с трудоустройством у аудитории 16–22 лет.
Frontend-часть обеспечивает удобный, современный и интуитивно понятный интерфейс для получения рекомендаций пользователя.
