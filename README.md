МТС Траектория — это информационная система с внедрённым AI-агентом, предназначенная для помощи в построении карьеры. Приложение решает ключевые проблемы, с которыми сталкиваются молодые люди на старте профессионального пути: отсутствие профессиональной ориентации, сложности в поиске подходящей работы и непонимание долгосрочного карьерного трека.
